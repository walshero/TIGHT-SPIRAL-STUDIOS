## BRIEF A — The four-skill measurement model *(BLOCKS THE BUILD)*

**The question:** Can a single decision-under-fog plus one reflection validly produce four distinct skill reads (leadership, problem-solving, critical thinking, media literacy) — or does it collapse into one blended signal wearing four labels?

**Who runs it:** the Measurement Design bench as a just-in-time expert pull — a psychometrician doppelganger (construct validity: is "four skills from one choice" a coherent measurement or noise?) and a decision-science doppelganger (does the literature on judgment-under-uncertainty support decomposing one decision into facets?). Grounded in real measurement literature via web research, the same way the v5 scale correction was grounded rather than guessed.

**What it must return (build-ready):**
1. A verdict: **independent scores** (four numbers) vs. **one construct with four facets** (one judgment read, shown four ways). The spec's own prediction is the latter — *judgment under incomplete information, with four facets* — but the bench confirms or overturns it.
2. The **honest-claim ceiling**: the exact sentence the "For instructors" panel is allowed to say about what the game measures. No over-claim. This is what ships in the file.
3. If facets: how each facet reads off the one decision (which beat, which signal) without pretending to be an independent instrument.

**Why this blocks the build:** the Beat 4 both-and read (consequence + mirror) is structured entirely by this answer. Build the mirror before the model is set and it gets rebuilt. This is the OS §10 ship-gate; it fires before code.

**Matt's HITL:** approve the honest-claim ceiling. That one sentence is a founder call — it's the promise the game makes to an instructor.

## BRIEF B — The JFK-1962 option set + fog *(BLOCKS THE JFK SLICE)*

**The question:** What are the 3–5 era-plausible postures a real advisor put on the ExComm table in October 1962, and the genuine unknowns the room faced — in 1962's vocabulary, with 1962's blindness?

**Who runs it:** a period-history bench (Cold War historian doppelganger) grounded in open, attributed sources — the declassified ExComm record is public domain (JFK Library, National Security Archive). This is a sourcing-and-history task, not a design guess.

**What it must return (build-ready):**
1. **The fog (Beat 1):** 3–4 genuine unknowns the room faced, each phrased in period vocabulary. The decisive one (historically, *"will Moscow read a blockade as an act of war"*) must be in the set but not flagged — the player has to find it.
2. **The postures (Beat 2):** the 3–5 options actually weighed — air-strike, quarantine, back-channel, invasion, hold — each in one period-true line, none flagged correct.
3. **Source tags:** every option and unknown carries an attributed open-license source (`data-art-source` schema), so the Fact-Check Lead can verify and the file can cite.

**Why this blocks the JFK slice:** Beats 1 and 2 *are* the leadership and problem-solving instruments. No slice without them.

**Matt's HITL:** does the option list feel 1962-true, or does it smell of 2026 hindsight? That's the Mad Men test, and it's the founder's eyes.

**Matt's HITL:** the register. This is voice work — 90% fidelity tier, Matt finishes.

## BRIEF D — The modern-three pairings *(gates the three modern scenes only)*

**The question:** For each modern president, which two sourced contemporary readings — balanced, attributed, open-license-where-possible — form the competing-records second frame?

**Who runs it:** the even-handedness bench + web research, selecting two real, dated, contemporaneous readings per scene that a fair observer would agree represent genuine competing frames, not a strawman and a favorite.

**Matt's HITL:** approve each pairing's balance. This is the highest-risk sourcing in the build — the even-handedness constitution governs it.

## BRIEF E — Consequence sourcing *(gates Beat 4 sourcing)*

**The question:** For each posture the player can choose, what did following that call actually meet — sourced, dated, honest?

**Who runs it:** the Fact-Check/Sourcing Lead + web research. Distinct from the v5 record-sourcing (which sourced *what the room couldn't see*); this sources *the consequence of the posture*.

**What it must return:** per scene, the honest consequence of the real historical call, attributed. For JFK: the quarantine held, Moscow withdrew, no shot fired — a genuine problem-solving triumph, sourced as one.

**Matt's HITL:** spot-check attribution. Mechanical, low-touch.
