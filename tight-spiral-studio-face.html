First — the principles

Everything in the studio runs on one loop and one rule. The loop is how a thing gets made. The rule is who stays in charge.

Play

Start by making the smallest real thing and using it. Never theorize in the abstract — improvement only comes from contact with actual work.

Notice

Watch what the work reveals — a friction, a gap, a reader who got lost. The honest catch matters more than the clever idea.

Design

Refine, then tighten. Adding requires pruning: every new rule must earn its place by letting something else be removed.

The rule under all of it: a human stays in the loop. The machine does the typing; the person keeps the judgment. The work is built to show its reasoning, not hide it — so it can be judged the way you'd judge a student's. A system that grades its own work, unwatched, is the exact thing this studio refuses. Accessibility is the floor, not a feature: build for the person who struggles most to see the screen, and the thing gets better for everyone.

Then — the tools

The Assignment Auditor

Drop in an assignment; it follows the river — what it teaches, how it's assessed, which rubric — and shows where the pieces line up before any student work is scored. The on-ramp to assessment.

The studio's own front-door sorter: name what you want from an idea, and it sorts it — build now, park, delegate, or cut — using the studio's verdict logic.
