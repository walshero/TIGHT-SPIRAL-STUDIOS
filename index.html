The Teaspoon

A game that is a paper · offline · single file · every fact is a tagged kernel

You are standing in a kitchen, holding a teaspoon. It is the smallest measure on the rack. Everything larger is built from it. Someone once checked exactly what it holds, wrote it down, and signed it — so nobody ever has to guess again.

Comfort is a knob, never a wall.
