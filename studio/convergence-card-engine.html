How to run one card

  1. State the question. One claim with real stakes.
  2. Gather independent sources. Different methods beat more of the same — that independence is the signal.
  3. Run the four seats. Denzin codes each source agree / partial / clash / silent. Cochrane weights by method quality, not volume. Marcus assumes the fluent answer is wrong until convergence proves it. Pearl asks what "works" even means for this claim.
  4. Sort into tiers. Converged / Contested / Thin / Silent. The Silent row is often the finding.
  5. You verify. Tap Verify or Reject on each claim. The panel advises; only you close.
Why this beats a single confident answer

A single answer gives you the Converged row and hides the rest. The Contested, Thin, and Silent rows are where a confident machine misleads you. Convergence across independent methods — not more sources, independent ones — is what makes this more trustworthy than one authority or one peer panel. In a 2026 agentic-verification study, machine verdicts matched two blinded human verifiers 91.3% of the time and an independent search engine 98.7%: the pairing is measurable, and it holds.
