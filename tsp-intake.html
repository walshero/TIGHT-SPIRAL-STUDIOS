Name it before you build it.

Three named things turn a pitch into a brief the fidelity gate can check the build against. If you can't name all three, it isn't ready to build.

The image The verb The source
What is this game called for now.
The one image or moment the first screen is about. Concrete, not a mood. Name the thing a player sees, not the feeling.
What the player can now DO that they couldn't before. One verb. Reachable in two taps.
The named, open-license text or domain the content rests on. Author, license, link. Averaged-internet fill is a failure, not a source.
One decision per screen is the floor. More than one halts.

Is this a reskin?

The engine repeats on purpose — Writerly Moves is the origin, every game since is the same mechanic in a new domain. A reskin is a new work only if the domain genuinely changed it. Same words, made new by where they stand. If nothing changed but the paint, it is a photocopy, not a new work.
